# 📋 COMList

> Registry tweak for **COM (Component Object Model)** configuration on Windows.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📋 **Registry Merge** | Double-click `.reg` to apply changes |
| 🔧 **COM Settings** | Modifies COM/CLSID-related registry keys |

---

## 📋 Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | Windows 10/11, Server |
| **Privileges** | Administrator |

---

## 🚀 Usage

1. **Create a restore point** (recommended)
2. Double-click the `.reg` file or right-click → **Merge**
